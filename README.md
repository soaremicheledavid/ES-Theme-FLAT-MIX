# ES-Theme-FLAT-MIX

Its a flat design, simple theme, lightweight, made for handhelds with 4/3 screens, high contrast menu, for easy of use.

# Aspect Ratio :
- Made for 4/3 screens, 640x480 resolution,  Handheld Console
  
# Gamelist View Style :
- BASIC
- DETAILED
- GRID
  
# Backgrounds :
- 7 WALLPAPERS
